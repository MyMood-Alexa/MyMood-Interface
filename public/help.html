How to Use Our Database

  1. Login to our database with valid credentials.
  2. Having granted access, you can view a table of available shared sessions between users.
      • If you have trouble finding a specific user, use the search bar to filter out any information about that user or session.
  3. To view a session, click on the box of the session ID that you wish to view.
  4. Viewing a session will provide the responses of the user throughout the session.

Attributes

Session ID: Identifier that uniquely identifies the session between the user and Alexa.
User ID: Identifier that uniquely identifies the user for a specific session.
Response: The statement recorded by the user or Alexa.
Time: The time that was recorded when a user started a session.

Frequently Asked Questions

Q: I forgot my username/password. How can I receive my account information?
  A: To receive your account information, contact us providing your registered email. If the email is valid, the account information will be directly sent to you.
Q: How frequently is the database updated?
  A: The database will be updated every Monday at Midnight.
Q: What can I do with this JSON file that I downloaded?
  A: This file is for your own benefit. You can simply use this file to save the session information or share these sessions with others.
